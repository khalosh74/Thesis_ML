# thesis_final_v1

Official thesis release bundle authority.

- `release.json`: bundle root and subordinate file references
- `science.json`: frozen scientific contract
- `execution.json`: run-class and runtime policy
- `environment.json`: environment policy for official release path
- `evidence.json`: artifact + verification contract
- `claims.json`: claim scope/boundaries

This bundle is candidate-only at run time. Official outputs are created only through promotion.
